'Hello World!'

 Donc la question est de décrire Git avec ses propres mots...à quelqu'un qui ne sait pas ce dont il s'agit.

Git est un outil puissant pour travailler sur des projets qui demandent du sérieux et de l'organisation. 
Il s'agit d'un outil qui nous permet de garder un historique détaillé de l'évolution du contenu de notre travail d'écriture au cours du temps.
Git nous permet de tracker les modifications apportés à notre ou nos fichiers et nous demande de commenter pour l'évolution pour archive.

Je ne pense pas que ce système dit de versionage soit réservé uniquement aux developpeurs, ça peut trouver son utilité pour bien d'autres activités, mais le développeur concentre de manière intense les besoins auxquels Git réponds.
En effet, un projet de developpement est quasi synonyme d'une quantité de données écrites astronomique, on y travail souvent en équipe et une petite erreur d'inattention qui ressemble à une aiguille dans une botte de foin peut tout bloquer ou corrompre.
